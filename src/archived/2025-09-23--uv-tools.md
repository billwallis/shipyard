- [BW 15 mins] uv tools ("brew for Python")
    - Canonical examples: [Ruff](https://github.com/astral-sh/ruff), [ty](https://github.com/astral-sh/ty)
    - Typical examples: [pre-commit](https://github.com/pre-commit/pre-commit), [Poe the Poet](https://github.com/nat-n/poethepoet)
    - Personal examples: [SQL lineage](https://github.com/billwallis/bills-sql-lineage), [data-diff](https://github.com/billwallis/data-diff)
    - *Installation instructions*
        ```bash
        uv tool install ruff
        uv tool install ty

        uv tool install pre-commit
        uv tool install poethepoet

        uv tool install bills-sql-lineage
        uv tool install https://github.com/billwallis/data-diff.git

        uv tool list
        ```
